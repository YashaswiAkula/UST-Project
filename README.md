<h1>Customer Support Ticketing System<\h1>
<h3>Overview<h3>
<h5>The Customer Support Ticketing System is a desktop application designed to streamline customer support operations. It provides a simple and intuitive interface for users to create, manage, and track support tickets.<h5>
<h3>Key highlights:</h3>

<h3>User Management:<h3> <h5>Allows registration and login with role-based access (Admin and User).<h5>

<h3>Ticket Management:<h3> <h5>Users can submit tickets, while admins can view, update, and close them.<h5>

<h3>Desktop GUI:<h3> <h5>Built using Tkinter, providing a user-friendly interface without needing a web browser.<h5>

<h3>Database Integration:<h3> <h5>Uses SQLite for storing all user and ticket information locally, ensuring lightweight and fast operations.<h5>

<h3>Easy to Use:<h3> <h5>Designed for small to medium-sized teams or businesses that want a simple, offline ticket tracking system.<h5>
